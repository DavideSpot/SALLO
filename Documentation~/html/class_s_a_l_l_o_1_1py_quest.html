Detailed Description

Psychophysical method component. Implements the QUEST adaptive procedure .

C# wrapper class. Internally, it calls the methods the QuestClient python class exposes, see the ./QuestZClient.py Python file for more details. It uses the Quest python class imported from the VisionEgg python package The communication between Unity and Python relies on the Unity Package Python for Unity
