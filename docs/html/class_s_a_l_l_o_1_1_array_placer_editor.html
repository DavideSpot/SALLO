override void OnInspectorGUI ()
 Add two buttons to the ArrayPlacer default inspector GUI:

  • Get Array calls the MakeList method, to import the available items in the array under control of the ArrayPlacer instance.
  • Update Positions calls the SetPositions method, to update the position of the items according to the current parameters set.
